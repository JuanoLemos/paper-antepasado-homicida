# On the inevitability of a homicidal ancestor

*Paper and reproducible code: why it is practically impossible not to have a
homicidal ancestor. A genealogical–probabilistic argument (pedigree collapse +
identical ancestors point) with a Monte Carlo simulation.*

[![DOI](https://zenodo.org/badge/1277539649.svg)](https://doi.org/10.5281/zenodo.20806914)

**Languages:** [Español](README.md) · English

A preprint (arXiv style) that formalises and quantifies the following claim:

> Every living human being has, with probability essentially equal to 1, at least
> one ancestor who caused the death of another human being.

The argument combines two facts:

1. **Exponential growth of ancestors** and **pedigree collapse**: the number of
   genealogical ancestors grows like `2^n` per generation until it saturates the
   breeding population of each era.
2. **Identical ancestors point (IAP)**: beyond a certain horizon (~5,000–7,000
   years; Rohde, Olson & Chang, 2004), the set of ancestors of any living person
   coincides with *the entire* global breeding population of that time. A single
   killer in that pool suffices to be an ancestor of everyone.

It also analyses **isolated populations** (islands, settlements founded by a few
colonists): the founder effect brings the IAP forward (`~1.77·log₂(N_e)`
generations) and shows why isolation is not an escape route from the thesis. It
also verifies **robustness to the independence assumption**: introducing
heritability/correlation `h` of violence (preserving the marginal prevalence),
the conclusion holds (it barely delays it by one generation).

## Structure

```
paper/        LaTeX documents (main.tex ES, main-en.tex EN) and bibliography (refs.bib)
  figures/    Figures generated by code/figures.py (ES and *_en EN)
code/         Model, figures and simulation in Python
  model.py    Core formulas (naive model, collapse, IAP)
  figures.py  Generates the paper figures
  simulate.py Monte Carlo simulation of ancestry + homicide
  results.py  Tables and numbers reported in the paper
  output/     Result CSVs (generated)
```

## Reproduce

```powershell
# 1. Python dependencies
pip install -r code/requirements.txt

# 2. Generate results and figures
python code/results.py
python code/figures.py
python code/simulate.py

# 3. Compile the PDFs (requires Tectonic)
tectonic paper/main.tex      # Spanish
tectonic paper/main-en.tex   # English
```

## Key definitions

- **Killer / homicide**: any person who caused the death of another human being
  (including war, self-defence, etc.). A broad and deliberate definition that
  maximises the robustness of the result.
- **Ancestor**: *genealogical* (lineal) ancestor, not genetic.
